# SubCareers Wiki

SubCareers 是一個 Minecraft RPG 副職業系統插件，基於 MMOCore + MMOItems + MythicLib + MythicMobs 建構。

提供 **9 種副職業**（伐木、耕作、釣魚、採礦、草藥、煉金、鍛造、刻印），搭配自訂 GUI、收集圖鑑、里程碑系統、MySQL/YAML 雙重資料儲存，以及獨立的農場空島系統與強化/升星/礦物融合等進階系統。
