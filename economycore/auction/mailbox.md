# 信箱

拍賣行的信箱是**離線也能領取**的物品收件匣。**信箱只裝物品**——賣出所得的銅錢會透過 Vault 直接入帳（離線也有效），不經過信箱。

## 查看與領取

- `/ah mailbox`：列出未領取物品，每筆可點擊領取。
- `/ah claim <信箱ID>`：領取指定物品（也可從 `/ah mailbox` 直接點。）

領取時若背包已滿，溢出的物品會掉落在你腳邊。

## 物品何時進信箱

| 原因（reason） | 情境 |
|----------------|------|
| `AUCTION_WON` | 競拍得標，物品送給得標者 |
| `BUYOUT` | 一口價購得，物品送給買家 |
| `LISTING_EXPIRED` | 競拍到期無人得標 / 流標，物品退回賣家 |
| `LISTING_RETURNED` | 下架退回，或上架介面開啟中離線時的物品保全退回 |

## 冪等投遞

同一筆掛單、同一原因只會投遞一次（資料表以 `(source_listing, reason)` 唯一鍵保證），即使結算重跑或跨節點掃描也不會重複發物。

## 與結算的關係

競拍結算選定得標者後，物品才進得標者信箱；無人得標或全員賴帳則退回賣家信箱。詳見[結算與退款](/economycore/auction/settlement)。
