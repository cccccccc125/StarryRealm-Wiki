# 競標

對**競拍（AUCTION）**掛單出價。到期時由最高有效出價者得標並結算。

## 如何出價

- **GUI**：在掛單詳情頁點「出價」→ 開鐵砧輸入金額（預填最低可出價）→ 確認。
- **指令**：`/ah bid <掛單ID> <出價>`。

```
/ah bid 3f9c2a1b-... 1500
```

## 出價規則

出價必須同時滿足：

1. 掛單是**競拍**且狀態為 **進行中**、**尚未到期**。
2. 你**不是賣家本人**。
3. 你**未被禁拍**（賴帳記點達門檻者會被短期禁止出價）。
4. 出價額 ≥ `max(起標價, 目前最高價 + 最小加價幅度)`。最小加價幅度預設為 `1`（銅錢）。

> **出價不會凍結你的銅錢。** 競拍採「結算才收款」：到期結算選定得標者時，才會向得標者收款。因此請確保得標時帳戶有足夠銅錢，否則會被[記點禁拍](/economycore/auction/penalties)並由次高者遞補。

## 並發保護

出價以樂觀鎖（CAS）更新目前最高價：若同一瞬間有人也出價導致你的更新落空，系統會自動重試；競爭過於激烈時回報 `OUTBID`，請重新出價。

## 出價結果

| 結果 | 意義 |
|------|------|
| `SUCCESS` | 出價成功 |
| `NOT_FOUND` | 找不到該掛單 |
| `NOT_AUCTION` | 該掛單不是競拍 |
| `NOT_ACTIVE` | 掛單已結束 |
| `EXPIRED` | 競拍已到期 |
| `SELF_BID` | 不能對自己的掛單出價 |
| `BANNED` | 你目前被禁止出價（賴帳記點） |
| `BID_TOO_LOW` | 出價未達「目前價 + 最小加價幅度」 |
| `OUTBID` | 出價競爭激烈，請重試 |
| `DB_ERROR` | 系統錯誤 |

到期後如何選定得標者、賴帳如何順延，見[結算與退款](/economycore/auction/settlement)。
